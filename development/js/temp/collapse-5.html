• Create a car. Pass the color of the car as a parameter. If color is not specified, it should default to 'black'.
  • The car should have a honk method, which sets its honking property to true immediately, and sets its honking property to false after 1 second
  • The car should have a run method, which sets its running property to true.
  • We will be creating many thousands of car objects rapidly, so we need it to be as efficient as possible for memory usage and speed of creation.
